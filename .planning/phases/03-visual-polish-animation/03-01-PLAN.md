**BaseLayout.astro** -- Add an inline script in `<head>` AFTER the `<slot name="head" />` line and BEFORE `</head>`. This script must be `is:inline` to run synchronously before first paint:

```html
<script is:inline>
  if (!window.matchMedia('(prefers-reduced-motion: reduce)').matches) {
    document.documentElement.classList.add('js-animate');
  }
</script>
```

This script:
- Runs synchronously before first paint (is:inline bypasses Vite bundling)
- Checks `prefers-reduced-motion` -- only adds class if user allows motion
- Adds `.js-animate` to `<html>` element, activating the CSS hiding rules
- If JS is disabled, the script never runs, class is never added, all content stays visible (per D-16, D-17)
- If reduced motion is preferred, class is NOT added, all content stays visible (per UI-SPEC accessibility)

Also add `position: relative; z-index: 1` to `<main>` so it sits above the constellation background:

Change `<main>` to: `<main class="relative" style="z-index: 1">`
