The Fano plane is an equilateral triangle with vertices, edge midpoints, and centroid. 7 points, 7 lines (3 triangle sides, 3 medians, 1 inscribed circle). Per D-27: no labels, esoteric sigil style, gold on dark.

**Component interface:**

**SVG specifications (from UI-SPEC):**
- viewBox: "0 0 400 360"

- role: "img"
- aria-label: "Fano plane diagram showing the seven quaternionic subalgebras of the octonions"

**7 point coordinates (equilateral triangle, circumradius ~150, centered at 200,190):**
```
P1 = (200, 40)    -- top vertex
P2 = (62, 280)    -- bottom-left vertex
P3 = (338, 280)   -- bottom-right vertex
P4 = (131, 160)   -- midpoint of P1-P2 (left edge)
P5 = (269, 160)   -- midpoint of P1-P3 (right edge)
P6 = (200, 280)   -- midpoint of P2-P3 (bottom edge)
P7 = (200, 193)   -- centroid ((200+62+338)/3, (40+280+280)/3)
```

**7 lines with assigned gold shades:**
1. P1, P4, P2 (left side) -- stroke #D4AF37, straight line P1 to P2
2. P2, P6, P3 (bottom side) -- stroke #C49B30, straight line P2 to P3
3. P3, P5, P1 (right side) -- stroke #B8892A, straight line P3 to P1
4. P1, P7, P6 (median from top) -- stroke #A87424, straight line P1 to P6
5. P2, P7, P5 (median from bottom-left) -- stroke #DAC060, straight line P2 to P5
6. P3, P7, P4 (median from bottom-right) -- stroke #E8D088, straight line P3 to P4
7. P4, P5, P6 (inscribed circle through midpoints) -- stroke #8B6914

All straight lines: stroke-width 1.5px, opacity 0.85.

**Inscribed circle (line 7):** The circle passes through P4(131,160), P5(269,160), P6(200,280). Compute circumscribed circle of these 3 points:
- P4 and P5 have same y=160, so the center x = (131+269)/2 = 200
- Using P4(131,160) and P6(200,280): center is (200, y_c), radius r
- (200-131)^2 + (y_c-160)^2 = (200-200)^2 + (y_c-280)^2
- 69^2 + (y_c-160)^2 = (y_c-280)^2
- 4761 + y_c^2 - 320*y_c + 25600 = y_c^2 - 560*y_c + 78400
- 4761 + 25600 - 320*y_c = -560*y_c + 78400
- 30361 - 320*y_c = -560*y_c + 78400
- 240*y_c = 48039
- y_c = 200.16 (approximately 200)
- r = sqrt(69^2 + (200-160)^2) = sqrt(4761 + 1600) = sqrt(6361) = 79.76 (approximately 80)

So inscribed circle: cx=200, cy=200, r=80. Render as `<circle>` element with stroke #8B6914, fill="none", stroke-width 1.5.

**Glow filter (applied only to node circles):**
```xml

  </feMerge>
</filter>
```

**Node circles:** All 7 points rendered as circles with r=8, fill="#C4A35A", filter="url(#fano-glow)".

**Render order:** defs first, then lines (behind), then inscribed circle, then node circles (on top).
